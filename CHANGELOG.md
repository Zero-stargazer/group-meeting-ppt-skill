# Changelog

## 2026-07-15

- 交付标准升级为：论文 PDF → 可编辑、可直接用于中文组会的 PPTX。
- 不再使用统一页数；页面数量由论文的证据链与汇报时长共同决定。
- 图表采用真实原图或面板级裁切；禁止整页论文、正文段落或图注段落进入成品页面。
- 每张图在生成前记录面板范围、可读性与裁切检查。
- 全部页脚统一使用本次 PDF 的论文信息，避免前次任务的作者或期刊残留。
- 完成前必须执行规格校验、全页渲染检查和 PPT 溢出检测。

## Public package scope

该发布包只保留当前 PDF→PPT 主链、必要的引用/科学严谨性规范与核心回归测试；不包含旧式 JSON→固定页数 PPT 辅助脚本、历史样例输出、内部工作目录或真实论文材料。
