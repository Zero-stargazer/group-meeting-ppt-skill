# PPT Style Profile

Use this only when the user asks for visual slide guidance or future PPTX generation.

## Visual Direction

- 16:9 widescreen.
- Chinese font: Microsoft YaHei, HarmonyOS Sans, or another clean sans-serif.
- Prefer one idea per slide.
- Avoid dense full-paragraph slides; use figure + short explanation.
- Use dark blue, white, cyan, and warm orange for emphasis if no lab template is provided.

## Layout Patterns

### Paper Overview

- Left: paper title, authors, source, year.
- Right: one-sentence summary and why this paper matters.

### Method Route

- Use a horizontal or vertical 3—5 step flow.
- Each step should have a verb: design, prepare, measure, decode, validate.

### Figure Explanation

- Left or top: original figure crop.
- Right or bottom: “这张图回答什么问题 / 它支持什么结论 / 我还要核对什么”.

### Advisor Questions

- Use a checklist layout.
- Group questions by method, evidence, figure interpretation, and relation to my topic.

## Readability Rules

- Do not put more than one large multi-panel figure on a slide unless it remains readable.
- If a figure has A/B/C/D panels, explain panels separately.
- If a generated explanation is longer than the figure, shorten the explanation before shrinking the font.
